# FitLife – Fitness Awareness Website 💪

A responsive **Fitness Awareness Website** that helps users learn about fitness and maintain a healthy lifestyle.
The website includes **workout information cards, BMI calculator, calorie calculator, and a weekly workout planner**.

---

## 📌 Project Overview

FitLife is a beginner-friendly frontend project built using **HTML, CSS, and JavaScript**.
The goal of this project is to spread awareness about fitness and provide useful tools that help users monitor their health.

The website provides:

* Basic information about different workout categories
* Health calculation tools like BMI and calorie calculator
* A simple weekly workout planner

---

## 🚀 Features

### 🏋️ Fitness Categories

Informational cards explaining different types of workouts:

* Cardio Training
* Strength Training
* Flexibility
* Nutrition
* Recovery
* Workout Routine

---

### 🧮 BMI Calculator

Users can calculate their **Body Mass Index (BMI)** by entering:

* Height
* Weight

The tool calculates BMI instantly and displays the result.

---

### 🔥 Calorie Calculator

This calculator estimates **daily calorie needs** based on:

* Age
* Height
* Weight
* Activity level

---

### 📅 Weekly Workout Planner

A simple weekly fitness plan:

* Monday – Cardio
* Tuesday – Strength
* Wednesday – Yoga
* Thursday – Cardio
* Friday – Strength
* Saturday – Flexibility
* Sunday – Rest
