# 🧠 Personality Types Guide with Interactive Quiz

An interactive awareness website that explains personality types (Introvert, Extrovert, Ambivert) and includes a short quiz to help users discover their personality type.

---

## 🚀 Features

- 📖 Informational personality cards
- 👤 Name input with greeting popup
- 📝 Interactive quiz (4 questions)
- 🧮 Automatic result calculation
- 🎉 Result modal with personality type
- 📱 Fully responsive design
- 🎨 Clean and modern UI

## 🧩 How It Works

1. User reads about personality types.
2. User enters their name.
3. Quiz appears after clicking "Start Quiz".
4. User answers 4 multiple-choice questions.
5. Based on answers:
   - Introvert
   - Extrovert
   - Ambivert
6. Result is displayed in a popup modal.
