# 🏋️ Beginner Workout Planner

A simple **goal-based workout planner** that helps beginners start their fitness journey by generating a weekly workout plan and tracking their progress.

This project allows users to choose a fitness goal such as **Weight Loss** or **Muscle Gain**, view a structured **weekly workout plan**, and mark completed workouts to track their progress.

---

## 🚀 Features

* 🎯 **Goal Selection**

  * Choose between **Weight Loss** and **Muscle Gain**

* 📅 **Weekly Workout Plan**

  * Automatically generates a **7-day workout schedule**

* 🖼️ **Exercise Cards**

  * Each workout includes:
  * Exercise name
  * Image
  * Short description

* 📊 **Progress Tracker**

  * Mark workouts as completed
  * Track weekly progress

* 🧭 **Navigation Tabs**

  * Home
  * About
  * Goals
  * Progress

* 📱 **Responsive Layout**

  * Works on desktop, tablet, and mobile screens
