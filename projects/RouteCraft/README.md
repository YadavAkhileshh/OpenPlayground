# RouteCraft - Optimized Route Planner

An optimized route planner that helps travelers visualize and rearrange destinations efficiently.

## Features

- Add destinations by name (geocoded using OpenStreetMap)
- Visualize destinations on an interactive map
- Manually rearrange destinations with drag and drop
- Optimize route using nearest neighbor algorithm
- Clear all destinations

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Leaflet.js for mapping
- OpenStreetMap for geocoding and tiles

## How to Use

1. Enter a destination in the input field
2. Click "Add Destination" or press Enter
3. Repeat to add more destinations
4. Drag and drop list items to rearrange manually
5. Click "Optimize Route" for automatic optimization
6. View the route on the map

## Running the Project

Simply open `index.html` in a web browser. No server required.
