How to Use

Creating Objects

  • Circle: Click and drag to set size
  • Rectangle: Click and drag to set dimensions
  • Polygon: Click to add vertices, double-click to complete
  • Random: Click to spawn random shapes

Controls

  • Left Click + Drag: Create objects or drag existing ones
  • Right Click: Delete object under cursor
  • Space: Pause/Resume simulation

Physics Parameters

  • Gravity: Controls downward force (0-2x)
  • Friction: Surface resistance (0-1)
  • Restitution: Bounciness (0-1.2)
  • Time Scale: Simulation speed (0.1x-2x)
