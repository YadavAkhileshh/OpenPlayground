Key Features

6 View Modes

Standard Grid, Compact Grid, Large Cards, Masonry, Timeline, and List views to suit any browsing preference.

Auto-Save Preferences

Your view mode choice is saved locally and persists across sessions automatically.

Keyboard Shortcuts

Press 1-6 to quickly switch between view modes, or Shift+S to cycle sort modes.

Smooth Animations

Beautiful transitions between layouts with CSS3 animations that respect user preferences.

Dark Mode Support

All view modes include full dark mode support with automatic color scheme detection.

Accessibility

Full keyboard navigation, ARIA labels, and support for reduced motion preferences.

Available View Modes

Standard Grid

2-3 columns. Balanced layout with good info density.

Compact Grid

4-6 columns. Dense layout for quick browsing.

Large Cards

1-2 columns. Extended info & larger thumbnails.

Masonry

Pinterest-style. Dynamic heights for visual browsing.

Timeline

Chronological display with date markers and progression.

List View

Compact linear format. Great for scanning.

Implementation Guide

Step 1: Import the View Mode Manager

1 Add to index.html

<script type="module">
  import { viewModeManager } from './js/viewModeManager.js';
  window.viewModeManager = viewModeManager;
</script>

Step 2: Add CSS Link

2 Link the CSS file

<link rel="stylesheet" href="./css/viewModes.css">

Step 3: Update Your HTML

3 Add container for controls

<div id="view-mode-controls-container"></div>

Step 4: Use in Your Code

4 Programmatic usage

// Switch view mode
viewModeManager.setViewMode('masonry');

// Listen for changes
window.addEventListener('viewModeChanged', (e) => {
  console.log('New mode:', e.detail.viewMode);
});
