# ArtLens

AI analyzes and gives feedback on uploaded artwork.

## Features
- Artwork upload (image file)
- AI feedback (simulated)
- Art style suggestions

## Usage
1. Open `index.html` in your browser.
2. Upload an image of your artwork.
3. Click "Analyze Artwork" for instant feedback and style suggestions.

## Tech Stack
- HTML, CSS, JavaScript (Frontend only)

## Notes
- No backend required.
- Feedback and suggestions are simulated for demo purposes.

## Contribution
- Add your project to `projects.json` (not `index.html`).
