# 🇮🇳 Indian Historical Leaders Website

A responsive and interactive **Indian Historical Leaders informational website** built using **HTML, CSS, and JavaScript**.

This project showcases important leaders who played a key role in shaping India's history and freedom movement. The website includes interactive features like **search functionality, modal popups, and a historical timeline** to make learning more engaging.

---

## 📌 Project Overview

The **Indian Historical Leaders Website** is designed to present information about influential leaders who contributed to India's independence and development.

The website uses a **modern card-based layout with animations and interactive components** to improve user experience and make educational content more accessible.

---

## ✨ Features

### 👤 Leaders Gallery

Displays cards for famous Indian leaders including:

* Mahatma Gandhi
* Jawaharlal Nehru
* Sardar Vallabhbhai Patel
* Subhas Chandra Bose
* Bhagat Singh
* Lal Bahadur Shastri
* Rani Lakshmibai
* Bal Gangadhar Tilak

Each card includes an image, short description, and a **Read More button**.

---

### 🔍 Search Functionality

Users can search for leaders by name.

* Filters leader cards dynamically
* Displays matching results instantly
* Clears results automatically when input is removed

---

### 📖 Read More Modal Popup

Clicking the **Read More** button opens a modal popup containing:

* Leader information
* Historical contribution
* Additional details

This improves the educational value of the website.

---

### 📅 Freedom Movement Timeline

A timeline section highlights important historical milestones:

* **1915** – Mahatma Gandhi returns to India
* **1920** – Non-Cooperation Movement
* **1947** – India gains Independence
* **1950** – Indian Constitution adopted

The timeline helps users understand the progression of India's independence movement.
