# Cyber Safety Awareness Guide Website

## 📌 Overview
This project is an interactive cybersecurity awareness website designed to educate users about important online safety practices.

## 🔐 Topics Covered
- Phishing
- Strong Passwords
- Two-Factor Authentication
- Social Media Privacy
- Public WiFi Risks
- Malware & Viruses

## ✨ Features
- Responsive grid layout
- Modern security-themed UI
- Search functionality to filter topics
- Hover card animations
- Mobile friendly design

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript
