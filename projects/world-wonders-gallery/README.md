# 🌍 World Wonders Explorer

A responsive and interactive **World Wonders Explorer Website** built using **HTML, CSS, and JavaScript**.
This project showcases the **Seven Wonders of the World** along with an option to explore and submit other historical places.

The website is designed with a **modern card-based UI, smooth animations, and responsive layout** to create an engaging user experience.

---

## 📌 Project Overview

World Wonders Explorer is an educational website that helps users learn about some of the most famous architectural landmarks in the world.

Users can:

* Explore the **Seven Wonders of the World**
* Learn about their **history and locations**
* View them through **interactive cards**
* Submit additional famous places using a form

This project demonstrates **frontend development skills including layout design, animations, and responsive UI design**.

---

## ✨ Features

### 🏛 Seven Wonders Gallery

Displays all **Seven Wonders of the World** with images and details:

* Taj Mahal – India
* Great Wall of China – China
* Machu Picchu – Peru
* Colosseum – Italy
* Petra – Jordan
* Christ the Redeemer – Brazil
* Chichen Itza – Mexico

Each wonder is displayed using **modern animated cards**.
