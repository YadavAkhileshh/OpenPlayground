Invisible Maze is a memory-driven puzzle game built entirely using
HTML, CSS, and vanilla JavaScript.

The player is shown a maze briefly. After the preview ends, all walls
disappear visually, forcing the player to rely on memory and spatial
reasoning to reach the exit.

---

## Features

- Memory-based gameplay
- Keyboard-only navigation
- Multiple maze layouts
- Difficulty-based preview timing
- Fog-of-war mode
- Collision penalty system
- Move and error analytics
- LocalStorage-backed settings

---

## Controls

- Arrow Keys or WASD to move
- Restart button to reset the maze

---

## Technical Highlights

- Grid-based maze rendering
- Game state machine
- Collision detection
- Analytics tracking
- Responsive design
- No external libraries or frameworks
