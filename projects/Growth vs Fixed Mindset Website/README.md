# 🧠 Growth vs Fixed Mindset Website

An educational website designed to help users understand the difference between **Growth Mindset** and **Fixed Mindset**.  
The project presents concepts using interactive cards, a self-reflection quiz, and motivational quotes to encourage continuous learning and improvement.

---

## 🚀 Features

### 📚 Informational Concept Cards
- Growth Mindset explanation
- Fixed Mindset explanation
- How to develop a growth mindset
- Daily habits for self-improvement

### 📝 Self-Reflection Quiz
- Simple mindset quiz
- Helps users reflect on their learning attitude

### 💬 Motivational Quotes Slider
- Auto-sliding quotes
- Manual navigation buttons
- Encourages positive thinking and learning

### 🎨 Modern UI
- Responsive layout
- Card hover animations
- Clean navigation tabs
- Beginner friendly interface
