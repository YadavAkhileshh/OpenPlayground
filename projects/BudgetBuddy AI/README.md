# BudgetBuddy AI

BudgetBuddy AI is a frontend expense tracker with monthly insights and actionable budgeting tips.

## Features
- Add and categorize expenses with date and note
- Set and update monthly budget targets
- Monthly KPIs: spent, remaining, usage, and spending forecast
- Category-wise spending chart
- Smart tips based on current spending behavior
- Expense history view
- JSON export and localStorage persistence

## Run
Open `index.html` in your browser.
