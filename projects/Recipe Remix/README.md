# Recipe Remix

AI suggests creative recipe variations based on your pantry.

## Features
- Pantry input
- AI recipe generator (simulated)
- Save favorites (localStorage)

## Usage
1. Open `index.html` in your browser.
2. Enter pantry items (comma-separated).
3. Click "Generate Recipe" for suggestions.
4. Save your favorite recipes.

## Tech Stack
- HTML, CSS, JavaScript (Frontend only)

## Notes
- No backend required.
- Favorites are saved in your browser's localStorage.
