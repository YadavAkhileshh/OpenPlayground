# EventNest

**Project Type:** Frontend (HTML/CSS/JS)
**Category:** Utility

## Description
EventNest is an event discovery and RSVP platform for local meetups and college communities.

## Key Features
- Event discovery by search and filter
- RSVP to events with confirmation
- Event history and local RSVP storage
- Responsive and accessible UI
