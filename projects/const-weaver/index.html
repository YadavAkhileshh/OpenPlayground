📖 Graph Theory Manual

The Algorithm (Kruskal's MST):

A Minimum Spanning Tree (MST) connects all vertices in a graph without any cycles and with the minimum possible total edge weight.

  • Disjoint-Set (Union-Find): The engine uses this data structure to instantly detect if a new connection will create a cycle. If it does, the connection is rejected!
  • Edge Sorting: The "Auto-Weave" feature gathers all possible valid connections, sorts them by distance, and iterates through them to build the mathematically perfect constellation.

How to Play:

  1. Click and drag between the glowing white stars to connect them.
  2. The length of the line costs "Stardust". Try to beat or match the Optimal Target!
  3. Dark purple Black Holes will sever any connection that passes through them.
