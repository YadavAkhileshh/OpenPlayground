Getting Started

Prerequisites

  • GitHub account
  • Git installed locally
  • VS Code recommended
  • Basic HTML, CSS, JavaScript knowledge

Setup

git clone https://github.com/YOUR_USERNAME/OpenPlayground.git
cd OpenPlayground

git checkout -b feature/your-project-name

npx live-server --port=8080

Adding a New Project

Step 1: Create Folder

mkdir projects/your-project-name
cd projects/your-project-name

Step 2: Required Files

  • index.html - Main file
  • style.css - Styles
  • script.js - JavaScript logic

Step 3: Create project.json ‚≠ê

{
  "title": "Your Project Name",
  "category": "utility",
  "difficulty": "Beginner",
  "description": "Brief description of your project (max 100 characters).",
  "tech": ["HTML", "CSS", "JavaScript"],
  "icon": "ri-code-s-slash-line"
}

Step 4: Test Locally

  • Open http://localhost:8080
  • Go to Projects section
  • Verify your project appears
  • Check console for errors

Step 5: Commit & Push

git add projects/your-project-name/
git commit -m "Add: Your Project Name"
git push origin feature/your-project-name

Step 6: Create Pull Request

  • Go to GitHub fork
  • Click Compare & pull request
  • Add screenshots
  • Submit PR

Project Guidelines

Technical Requirements

  • Use vanilla HTML, CSS, JavaScript
  • Make it responsive
  • Test across browsers
  • Write clean code

Quality Standards

  • No errors
  • Good UX
  • Optimized images
  • Accessible content

Content Guidelines

  • Original work only
  • Educational or fun projects
  • No harmful/offensive content
